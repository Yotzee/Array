## C++ Dynamic Array


### Creates array of size 200 and will increase size every 200
```
	Array<int> array;
```

### Creates array of size 500 and will increase size every 200
```
	Array<int> array = Array<int>(500);
```

### Add Object
```
	array += 1; // add number
```

### Remove Object at Index
```
	array.remove(0);
```

### Get Object at Index
```
	array[1]
```
